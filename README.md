## Laravel Version

Version 11

## Using 

1. Clone this repository
2. Composer install
3. cp.env .env example
4. php artisan key:generate


## Database Migration

1. make database in the .env
2. make database in local server
3. php artisan migrate (from terminal)

## User Account
